Donde iniciar Laravel

Para iniciar en laravel como en todo lo que tiene que ver en la programación no hay una ruta especifica, sin embargo, se darán 3 lugares donde comenzar adquirir el conocimiento de este framework y todo lo relacionado

Sitios para el aprendizaje

coders free

En este sitio podrás aprender todo lo referente a laravel en español, además podras crear proyectos que servirá como base para el portafolios, aparte hay una seccion de tips que ayudará a resolver algunas dudas que por lo general se prensentan cuando uno esta empezando en este framework.

coders free

laracast

Este sitio permite ampliar de manera significativa los conocimientos en el framework laravel así, como sus multiples integraciones y paquetes destinados a la escalabilidad y progresión de las plataformas web desarrolladas con laravel.

laracast

laravel docs

La documentación es parte fundamental de todo lenguaje de programación y para este framework no es la excepción, laravel cuenta con una amplia documentación que permite por si solo enteder el framework y empezar a trabajar con el de una manera muy intuitiva.
