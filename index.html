The PHP Framework for Web Artisans

Laravel es un framework de aplicaciones web con sintaxis expresiva y elegante. Ya hemos sentado las bases, liberándote para crear sin sudar las pequeñas cosas.

Por qué laravel?

Un marco progresivo

Nos gusta llamar a Laravel un marco "progresista". Con eso, queremos decir que Laravel crece contigo. Si recién está dando sus primeros pasos en el desarrollo web, la vasta biblioteca de documentación, guías y tutoriales en vídeo te ayudará a aprender las cuerdas sin agobiarte.

Un marco escalable

Laravel es increíblemente escalable. Gracias a la naturaleza amigable con el escalado de PHP y el soporte incorporado de Laravel para sistemas de caché rápidos y distribuidos como Redis, el escalado horizontal con Laravel es muy fácil. De hecho, las aplicaciones de Laravel se han escalado fácilmente para manejar cientos de millones de solicitudes por mes.

Un marco comunitario

Laravel combina los mejores paquetes en el ecosistema PHP para ofrecer el marco más robusto y amigable para el desarrollador disponible. Además, miles de talentosos desarrolladores de todo el mundo han contribuyeron al marco.
